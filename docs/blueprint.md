# **App Name**: Farm Mart

## Core Features:

- Multi-Role Authentication System: Secure role-based access for Farmers, Customers, Investors, and Business Partners with custom redirect logic and session persistence.
- Live Panen Commerce: Interactive live-streamed harvesting interface allowing customers to buy fresh produce in real-time with integrated pinned products and comments.
- Farmer SaaS Dashboard: A comprehensive analytics and product management portal for farmers to track sales, manage inventory, and monitor market price trends.
- Investor Portfolio Tracker: A financial dashboard for agricultural investment, featuring ROI charts, funding progress, and sustainability impact analytics.
- Harvest Prediction Tool: A generative AI tool that uses historical crop data and weather patterns to advise farmers on the best harvest windows and yield optimization.
- B2B Wholesale Hub: Dedicated interface for HORECA partners to manage bulk orders, Requests for Quotations (RFQs), and recurring agricultural supplies.
- Persistent Data Storage: Save and retrieve marketplace products, user profiles, and order history using a decentralized local and cloud database system.

## Style Guidelines:

- Primary Colors: A sophisticated mix of Organic Green (#2E7D32) for freshness and Zesty Orange (#F57C00) to represent harvest energy and ripeness.
- Secondary color: Leaf Green (#43A047) used for interactive elements and success states.
- Background: Soft Cream (#F7F8F4) to create a warm, premium editorial feel across all dashboards.
- Accent: Harvest Gold (#F4B400) for high-impact call-to-actions and warning states.
- Headline and body text: 'Poppins', a geometric sans-serif that ensures a precise, modern, and fashionable appearance.
- Clean, thin-stroke agricultural and financial icons to maintain a sophisticated SaaS aesthetic across all role dashboards.
- Editorial-driven storytelling layouts using glassmorphism components, floating cards, and generous white space inspired by premium marketplaces.
- Silky-smooth transitions, hover animations on orange/green buttons, and ripple effects to reinforce the premium nature of the platform.
